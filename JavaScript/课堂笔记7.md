> 简单数据类型在内存的存储方式和赋值方式。
```js
    简单数据类型是在栈区保存的结果
    简单类型的赋值过程是将 一个值 拷贝一份 赋值另外一个变量
```
> 复杂数据类型在内存中的存储方式和赋值方式。
```js
    复杂数据类型是堆区中保存的结果
    复杂数据类型在赋值的过程中，将内存地址 拷贝了一份 赋值给另外一个变量。
```

> Math对象
```js

    Math.floor()

    Math.ceil()

    Math.random()

    Math.PI  ---> 圆周率

    Math.pow(x, y);    ---> 计算 x 的 y 次方

    Math.abs(x)   --->  获取当前数字绝对值

    Math.round(X)  ---> 对数字进行四舍五入后的整数

    Math.max()  ----> 用来获取一组数字中的最大值。  

    Math.min()  ----> 用来获取一组数字中的最小值。

    Math.sin(a) ---> 获取角的正弦值

    Math.cos(x)   ---> 获取角的余弦值
   
```
> Date对象
```js

    日期对象：

         var  d = new Date();


    其他写法：
         1. 可以给Date()传递参数

            ✔ 可以传递表示时间的字符串

             var d = new Date('2019-8-17 18:20');

            ✔ 可以传递表示时间的数字

            var d = new Date(2019, 8, 17, 18,20);

            ✔ 可以传递用来表示时间的毫秒数

                 日期对象.valueOf();

                 日期对象.getTime();

                 Date.now();     ---- H5中的一个方法，存在兼容性

```
> 数组对象
```js
    var  ary = new Array()

    ☞ 取值

        数组名.pop()        从数组中最后位置取值

        数组名.shift()      从数组中开始位置取值
        
    ☞ 赋值

        数组名.push();              在数组的结尾处添加一个值

        数组名.unshift(number);     在数组的开始位置添加值

    ☞ 筛选

         indexOf():     获取数组中对应值的索引位置，如果找到那么直接返回对应的索引值，如果没有找到，返回 -1
         
         lastIndexOf() : 获取数组中对应值索引的位置，从后向前找，如果没有找到返回 -1

    ☞ 翻转

          数组名.reverse()

    ☞ 拼接字符串

         数组名.join('分割符');   ====》 返回的是一个字符串

```
> 字符串
```js

    ☞ 字符串.split('分隔符')  ===》 数组

            如果是负数，那么从最后做一个减法

    ☞ 去掉空白

        str.trim()   ----> 去掉字符串中的首尾处的空白
```

> 静态成员 和实例成员
```js
     静态成员：  该对象中的方法或属性不需要通过构造函数创建对象就能直接访问


     实例成员：  对象中的方法或属性需要通过构造函数创建对象后，才能访问     
```
