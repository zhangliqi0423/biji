# 函数的其他部分
> 函数的另外一种定义方式
```js
    ☞ 通过表达式的方式定义函数
    var 自定义名称 = function() {}

    ☞ 函数的执行
       自定义名称()
```
> 匿名函数
```js
     匿名函数： 函数没有名字

        function () {}

     命名函数： 函数有名字
        function fn() {}
        fn()
```
> 自调用函数
```js
    自调用函数（自执行函数）：函数自己调用自己。
    (function(){})()
```
> 函数中的arguments。
```js
    arguments: 在函数中用来保存实参信息的一个容器

        ☞ 实参的个数:  length( 保存的就是实参的个数)
    
        ☞ 实参中的具体值： 值是以数组的形式保存在arguments中的。

    总结：
         1. 如果函数中的形参个数是确定的，那么推荐使用形参获取即可

         2. 如果函数中的形参个数不确定，那么可以通过arguments的方式获取到
```
> 代码预解析
```js
     
    代码预解析： 在程序开始执行之前，将代码进行一个预先编译的过程

        1. 变量的提升：当程序中出现变量，那么就会将变量的声明提升到当前作用域的开始位置，不包括变量的赋值。

        2. 函数的提升：当程序中出现函数，那么就会将函数的声明提升到当前作用域的开始位置，不包括函数的调用。
```

# 对象
> 什么是对象？
```js

    对象： 在程序中对具体事物的一个抽象描述（特征， 功能）。 本质上： 对象还是一个容器

```
> 学习对象
```js
    ☞ 学会如何定义（声明）一个对象

         1.  通过字面量的方式定义一个对象

             var 自定义对象名称 = {};

    ☞ 如何给对象赋值

        1. 定义对象的时候就可以赋值（初始化）

                var zs = {
                
                    //  键值对
                    username : '张三',

                    height : 180,

                    weight : 75,
                    //张三会唱歌
                
                    sing : function() {

                        alert('张三正在陶醉的唱着： 社会摇...');
                    },
                    //张三会rap

                    dance : function () {
                        alert('张三正在愉快的摇摆...');
                    }
                }
        
        2. 通过键值对的方式给对象赋值。

               var obj = {};

                // obj['键名'] = 值

                obj['username'] = '张三';

                obj['sing'] = function() {
                    alert('张三正在唱歌');
                }

        3. 通过 '对象.'的方式给对象赋值。

                var obj = {};

                obj.username = '张三';

                obj.height = 180;

                obj.sing = function() {
                    alert('张三正在唱歌');
                }

        注意：
               ✔ 对象中的值都是以 键/值 对的形式出现。

               ✔ 键： 就是一个自定义的变量

               ✔ 在对象中的函数叫方法  ---》 实现一个具体的功能

               ✔ 在对象中用来描述对象的特征的键，称为对象的属性。

               ✔ 在函数中如果定义变量没有使用var关键字，那么该变量是全局变量

    ☞ 如何从对象中取值

          1. 通过键值对的方式取值

             例如：  zs['sing']();     zs['username']


          2. 通过 '对象.'的方式获取对象中的值

             例如：  zs.username    |  zs.dance();

```

# 其他创建对象的方式
>  通过构造函数创建对象
```js
   
     var 自定义对象名称 = new Object();

      1. Object()  --- 是js中已经内置好的一个构造函数（本质还是函数），实现创建对象的。

      2. new  --- 是一个关键字，new关键字就是用来调用构造函数创建对象的。


      总结：
         1. 通过对象的方式创建数组
           var  ary = new Array();
```

>  通过工厂的方式创建对象
```js

     本质： 就是将构造函数创建对象的方式封装成一个函数

     例如：
       function creatObject(uname, uage) {

            var people = new Object();
                people.username = uname;
                people.age = uage;
            return people;
        }
        var zs = creatObject('张三', 20);

        var ls = creatObject('李四', 30);
```

> 通过自定义构造函数的方式创建对象
```
     步骤：
         1. 先准备一个自定义的构造函数


         2. 通过自定义构造函数创建对象


         例如：

         function  People(uname, userage) {

              this.username = uname;
              this.age = userage;

              //唱歌的方法
              this.sing = function() {
                  alert('正在唱歌');
              }
          }

          // 通过构造函数People 创建了一个对象 zs
          var zs = new People('张三', 22);
          var ls = new People('李四', 25);
          var ww = new People('王五', 28);


     注意：

         1. 首先构造函数的命名规范是：帕斯卡命名法（大驼峰命名法）[每一个单词的首字母都要大写]

         2. 构造函数是以名词开始的

```
